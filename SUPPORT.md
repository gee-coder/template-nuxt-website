# 支持说明

## 1. 仓库定位

本仓库是创业技术底座中的 Nuxt 3 官网模板，默认提供营销首页、能力介绍、关于我们与联系咨询等公开页面能力。

该仓库提供模板级支持，适合快速搭建官网基座，不承诺生产 SLA。

## 2. 提问前建议先检查

- [README.md](./README.md)
- [CONTRIBUTING.md](./CONTRIBUTING.md)
- [SECURITY.md](./SECURITY.md)
- [RELEASE.md](./RELEASE.md)
- `nuxt.config.ts`、运行时配置与联系表单接口约定

## 3. 适合提交公开 Issue 的场景

- 页面渲染、SEO、路由、联系表单等主流程存在可复现问题
- 文档、运行时配置或部署说明与实际行为不一致
- 通用官网模板能力有沉淀价值的增强建议
- Nuxt 或依赖升级引发兼容性回归

## 4. 不建议提交公开 Issue 的场景

- 安全漏洞、配置泄露、表单滥用或生产环境敏感信息
- 带有真实客户数据、业务投放参数或内部部署拓扑的内容
- 明显属于单个品牌项目专用的定制需求

安全相关内容请改走 [SECURITY.md](./SECURITY.md)。

## 5. 提问时建议包含的信息

- Node.js 与 npm 版本
- 部署模式：Node、静态托管或容器
- 最短复现步骤
- 页面截图、控制台报错、网络请求结果
- 运行时配置与后端接口约定

## 6. 响应优先级

- 可复现且影响公开页面主流程的问题优先
- 配置和部署文档错误优先
- 新增模块会结合模板通用性统一评估

## 7. Fork 使用建议

如果你将本模板用于商业官网，建议改写支持渠道、品牌信息、联系人与发布策略，避免后续治理文档和真实运营流程脱节。
