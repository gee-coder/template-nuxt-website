# template-nuxt-website

Nuxt 3 marketing website template for startup landing pages. It includes a homepage, feature page, about page, and contact page wired to the backend contact submission API.

## Stack

- Nuxt 3.21.2
- Vue 3
- TypeScript

## Start

```powershell
npm install
npm run dev
```

## Environment

Copy `.env.example` to `.env` and adjust the API base URL if needed.

## Pages

- `/`
- `/features`
- `/about`
- `/contact`
