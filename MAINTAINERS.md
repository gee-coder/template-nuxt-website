# 维护者说明

## 1. 当前维护角色

- 仓库所有者：`gee-coder`
- 默认维护范围：官网模板结构、页面模块、联系表单集成、文档与 CI

fork 后建议及时替换为自己的维护者、评审人与发布负责人。

## 2. 重点评审范围

- 运行时配置、SEO 与元信息策略
- 联系表单、公共内容模型与请求封装
- 主题样式、全局布局与组件抽象
- 核心依赖升级与构建链路调整

## 3. 合并标准

- `npm run type-check` 与 `npm run build` 通过
- 页面行为变更同步更新文档或配置说明
- 保持模板通用性，避免沉淀过强品牌绑定逻辑
- 对 SEO、性能与可维护性的影响有基本说明

## 4. 分支与版本建议

- 稳定分支：`master`
- 功能分支：`feat/*`
- 修复分支：`fix/*`
- 发布标签：`vX.Y.Z`

## 5. 发布责任

发布者需要确认：

- `CHANGELOG.md` 已更新
- `RELEASE.md` 中的检查项已执行
- 对接后端接口与部署说明保持同步
