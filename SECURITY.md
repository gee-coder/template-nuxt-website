# 安全说明

感谢你帮助提升模板仓库的安全性。

## 1. 适用范围

本仓库是官网模板，以下问题建议优先通过私下方式反馈：

- 联系表单可能导致信息泄露或滥用
- SEO、注入、运行时配置存在风险
- 页面脚本、依赖或构建配置存在高危问题

## 2. 不建议公开披露的内容

- 可直接利用的漏洞细节
- 真实环境密钥、凭证、接口白名单信息
- 攻击脚本或敏感配置截图

## 3. 推荐反馈方式

- 描述问题现象、影响范围、复现条件
- 不要在公开 issue 中贴敏感信息
- 优先联系维护者确认后再决定公开披露方式

## 4. 修复策略

维护者会优先修复高风险问题，并同步更新文档、默认配置或部署说明。
